Figure 6 code utilities

Scripts:
- fig6_cd_prepare_source_data.py
  - Extracts ../fig6c.gz and ../fig6d.gz
  - Exports Panel C/D source tables (dotplot/heatmap/violin)

- fig6_cd_dotplot.py
  - Quick replot helper for Panel C/D dotplots

- fig6_source_data.py
  - Unified builder for Figure 6 source data package (Panels A-I)
  - Generates Fig6_file_mapping.csv and Fig6_panels_data_availability.csv
