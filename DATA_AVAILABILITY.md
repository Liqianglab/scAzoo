# Data availability

Processed single-cell analysis data are deposited in OMIX under accession number `OMIX017477`.

This repository provides figure-level source data, supplementary tables, and analysis scripts. It does not include raw FASTQ files and does not represent a complete raw-read-to-figure workflow.

Large binary assets and source-package files that are unsuitable for ordinary GitHub upload are listed in `metadata/large_assets_for_lfs_or_release_manifest.csv` and `metadata/excluded_files_manifest.csv`.

For manuscript wording:

> Processed single-cell analysis data have been deposited in OMIX under accession number OMIX017477. Figure-level source data and analysis code are available at the project GitHub repository.
